Vê o status do git

git add "nome do arquivo.extenção"
Adiciona um arquivo ao git

git add .
Adiciona todas as pastas ao git

git -m commit "nome do commit"
Cria uma versão uma versão do programa

git remote add origin URL
Adiciona o local para upar

git push --set-upstream origin master
Upa para o github

git pull

AVANÇADO

git reset --hard ID
Reseta o git para alguma versão específica pelo ID

git branch
Vê os branches

git branch nome
Cria novo branch

git checkout nome
troca de branch

git checkout -b novo-branch branch-base
cria e muda para um novo branch criado com uma base em um existente

git pull
Puxa a atualização mais recente do servidor

git merge "nome"
Puxa as atualizações de um branch
